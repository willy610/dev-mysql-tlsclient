    Available collations mapped to the internal ID. Handshake packet have only 1 byte for collation_id. Only collations with ID > 255 are used during the handshake The list of collation is from this SQL query: SELECT ID, COLLATION_NAME FROM information_schema.COLLATIONS WHERE ID <= 255 ORDER BY ID;

    ucs2, utf16, and utf32 are excluded since they cannot be set as connection charset. https://dev.mysql.com/doc/refman/5.7/en/charset-connection.html#charset-connection-impermissible-client-charset
